Fun Dev Facts

  • The first computer bug was an actual bug - a moth found in a computer in 1947.
  • The first computer programmer was a woman - Ada Lovelace in the 1840s.
  • More than 90% of the world's currency exists only on computers.
  • The average person blinks 15-20 times per minute, but only 7 times when using a computer.
  • There are over 700 programming languages in existence.
